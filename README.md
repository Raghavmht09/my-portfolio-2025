# Portfolio Website

A modern, responsive portfolio website built from scratch.

## Getting Started

1. Open `src/index.html` in your browser to view the website
2. Customize the content in `index.html` with your information
3. Modify styles in `assets/css/style.css` as needed
4. Add functionality in `assets/js/main.js`

## Project Structure

```
port-web-app/
├── src/           # HTML files
├── assets/        # CSS, JS, and images
├── public/        # Build output (when needed)
├── docs/          # Documentation
└── CLAUDE.md      # Development guidelines
```

## Features

- Responsive design
- Smooth scrolling navigation
- Modern CSS with gradients
- Clean HTML5 structure
- JavaScript interactivity
